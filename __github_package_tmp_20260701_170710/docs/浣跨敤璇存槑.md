# 使用说明

## 输入字段

- Sorftime CLI Token：你的 CLI 授权 token，可临时填在网页中。
- 自有 ASIN：自己的产品 ASIN，每行一个。
- 竞品 ASIN：竞品产品 ASIN，每行一个。
- 核心关键词：用于查询关键词自然位、广告位、搜索量等，每行一个。

## 输出格式

- HTML 看板：适合开会查看和归档。
- PDF：适合发给团队或老板。
- Markdown：适合二次编辑、放进飞书文档或知识库。
- JSON：标准数据协议，方便后续接入 n8n、飞书多维表格或数据库。
- CSV：ASIN 明细表，可直接用 Excel 打开。
- ZIP：打包全部输出文件。

## 建议用法

第一天先输入 1 个自有 ASIN + 3 个竞品 ASIN + 5 个核心关键词，确认 CLI 调通。
确认没问题后，再扩大到 10-20 个竞品和更多关键词。
